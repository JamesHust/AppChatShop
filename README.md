# AppQuickShop
Quick order application in just 3 steps on smartphone with integrated online chat.


## Installation
###### Using npm
**Installation for backend**
```sh
cd .\backend
npm install
```

**Installation for frontend**
```sh
cd ..
cd .\frontend
npm install
```

###### Running project
**Running for backend**
```sh
npm start
```

**Running for frontend**
```sh
expo start
```
